# Social-Media-Influencer-Segmentation-for-Targeted-Campaign

## Overview
Social media influencers are crucial in promoting brands and engaging consumers in today's 
digital marketing landscape. Marketing campaigns can become significantly more effective by 
strategically selecting influencers to target specific audiences. This project utilizes machine 
learning and data analytics to segment social media influencers based on various impact 
metrics. We aim to create detailed influencer segments by analyzing Key Performance 
Indicators (KPIs). This will enable brands to refine their marketing strategies and improve 
campaign return on investment (ROI). 

## Key Findings
- Instagram and TikTok show similar cluster distributions, while YouTube is less utilized, highlighting a shift toward short-form content platforms.
- For region-based business expansion, companies should consider geographic targeting and select specific influencers within Clusters 2 and 3 for higher local engagement.
- Influencers in Cluster 1 have widely distributed audiences across regions, making them well-suited for broad campaigns such as product launches targeting a large audience.

## Data Usage
- `campaign`: contains records of campaigns involving influencer collaborations
- `influencers`: includes social media performance metrics of influencers across platforms used
